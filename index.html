Tag Description
<html> The root element of an HTML page.
<head> Container for metadata, CSS links, scripts, and the page title.
<title> Defines the title of the webpage (displayed in the browser tab).
<meta> Specifies metadata about the document (e.g., charset, author, viewport settings).
<link> Defines a relationship to an external resource, such as linking a stylesheet.
<style> Used to define internal CSS styles for the document.
<body> Contains the content of the HTML document.
<header> Defines a header for a document or section.
<nav> Defines a set of navigation links.
<section> Defines a section in a document.
<article> Defines independent, self-contained content.
<aside> Defines content aside from the main content, often in a sidebar.
<footer> Defines a footer for a document or section.
<main> Specifies the main content of a document (only one per page).
<div> Defines a division or section in a document.
<span> Defines a section in a document used for inline elements.
<h1> to <h6> Defines HTML headings from level 1 (highest) to 6 (lowest).
<p> Defines a paragraph.
<a> Defines a hyperlink.
<img> Embeds an image.
<ul> Defines an unordered (bulleted) list.
<ol> Defines an ordered (numbered) list.
<li> Defines a list item.
<table> Defines a table.
<tr> Defines a table row.
<th> Defines a table header cell.
<td> Defines a table cell.
<form> Defines an HTML form for user input.
<input> Defines an input control.
<button> Defines a clickable button.
<select> Defines a drop-down list.
<option> Defines an option in a drop-down list.
<textarea> Defines a multi-line text input control.
<label> Defines a label for an <input> element.
<br> Inserts a line break.
<hr> Inserts a horizontal rule.
<iframe> Embeds another HTML page within the current page.
<audio> Embeds an audio file.
<video> Embeds a video file.
<source> Defines multiple media resources for <audio> or <video>.
<canvas> Used to draw graphics via scripting (usually JavaScript).
CSS Property Description
color Sets the text color of an element.
background-color Sets the background color of an element.
width Specifies the width of an element.
height Specifies the height of an element.
font-size Sets the font size of text.
font-family Specifies the font of the text.
padding Generates space inside an element's border.
margin Generates space outside an element's border.
border Sets the border around an element.
display Specifies the display behavior of an element (block, inline, flex, etc.).
position Specifies how an element is positioned in a document (relative, absolute, fixed, sticky).
flex Configures flexbox layout for an element.
grid Configures grid layout for an element.
z-index Specifies the stack order of an element (higher value appears on top).
transition Defines the transition effect when a property changes.
